# 現代文記述添削アプリ 試作版

本文は表示せず、設問・入力欄・添削ボタンだけを表示するプロトタイプです。

## 動かし方

```bash
npm install
cp .env.example .env.local
# .env.local に OPENAI_API_KEY を入れる
npm run dev
```

OpenAI APIキーが未設定の場合も、簡易添削モードで動作確認できます。

## 画面仕様
- 作品選択：ミロのヴィーナス
- 設問選択：記述問題のみ
- 生徒答案入力
- 添削結果表示

## データ方針
- 生徒画面には本文を表示しない
- 模範解答・採点観点はサーバー側でのみ利用
- 将来的には Microsoft/Teams アカウントログインを追加する想定
